## Cách chạy chương trình

#### Clone:
cp .env.example .env
composer install
php artisan key:generate
php artisan migrate:fresh
php artisan db:seed
php artisan storage:link
npm install
npm run dev
php artisan serve

#### Pull:
composer install
php artisan migrate
php artisan serve
